# PDF Books Directory

This directory stores PDF versions of books available in the library.

## Structure

Place PDF files here with the naming convention: `{isbn}.pdf`

For example:
- `978-0-7432-7356-5.pdf` for book with ISBN 978-0-7432-7356-5
- `978-0-262-03384-8.pdf` for book with ISBN 978-0-262-03384-8

## Features

- **View PDF**: Users can view PDFs directly in the browser
- **Download PDF**: Users can download PDFs for offline reading
- **Access Control**: Only logged-in users can access PDFs

## File Requirements

- Format: PDF only
- Size: Recommended max 50MB per file
- Naming: Use ISBN number as filename (remove hyphens is optional)

## Sample Books

Add your PDF files here. The system will automatically detect them based on ISBN.
