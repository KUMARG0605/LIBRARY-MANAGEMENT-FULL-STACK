# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-11-29

### Added
- Complete Flask application with MVC architecture
- User authentication system (login, register, password reset)
- Book management (CRUD operations)
- Borrowing system with due dates and fine calculation
- Reservation system for unavailable books
- Review and rating system
- User dashboard with statistics
- Admin dashboard with analytics
- RESTful API endpoints
- Docker and docker-compose configuration
- Nginx reverse proxy setup
- PostgreSQL database support
- Redis caching integration
- Email notification system (Flask-Mail)
- Responsive Bootstrap 5 UI
- Mobile-friendly design
- Search and filter functionality
- Category and department management
- Activity logging
- System settings
- Error pages (404, 500, 403)
- Comprehensive documentation
- Setup and run scripts
